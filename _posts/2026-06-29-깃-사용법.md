jerlkey 테마 이게 맞나?
1. FORK
    fork시 내 repositories 생성
2. git clone "내주소"
    내 폴더에 파일 생성
3. bash tools/init.sh
    홈페이지 최신 init 설정
4. git add .
    깃 커밋(모든 파일을 사진 찍기) 직전, 무대 위에 올린다.
5. git commit -m "chore: complete initial chirpy setup"
    내 모든 파일을 사진 찍는다. 
6-1. git push origin master
    그 사진을 서버에 올린다!
6-2. git push origin master --force
    내 파일과 기존 파일이 꼬였을 때 강제로 올리기!
